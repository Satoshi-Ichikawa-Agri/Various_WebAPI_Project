## プロジェクト名

Various_WebAPI_Project

## プロジェクトについて

いろんな WebAPI を集めたプロジェクト
以下機能を実装した。

- 都道府県の自治毎に設置されている公営体育館を検索できる API
- お気に入りラジオの放送回を検索する API
